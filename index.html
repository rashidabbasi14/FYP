Features You Should Know!.

The reason behind making this project is to help in getting the Crime Rate be minimized by being able to track and catch the culprits.

01.

Tracking

Easy and user friendly visual representation of Culprit on Google Map. Showing exact location of suspect at the time he was found. Keeping trace of his moments.

02.

Accuracy

The application will display functionality which will be according to the user so that user does not confuse at any point that will increase the productivity and efficiency of the user.

Real Time

Seamless system providing real time functionality for the authorized authority to track unknown suspects using only single face image.
